Cambridge GIS Open Data on GitHub

The City of Cambridge is committed to building repositories for data to be used by developers and for public consumption. Cambridge GIS has posted the majority of the data sets on this official City of Cambridge site. Some larger size files which do not lend themselves to this site (such as elevation and raster data) are available for download on our Data Dictionary site.

GeoJSON and TopoJSON

GitHub recognizes our GeoJSON and TopoJSON files and creates interactive maps using Mapbox for most files. You can also view or download the raw GeoJSON or TopoJSON using the 'Raw' button.

Report Issues and Update Data

If you notice a problem with the data, you can report an issue on this site using the New Issue form.

You can also click 'Edit' on any GeoJSON file to make a fork/copy on your own account, make changes, and suggest that they get adopted here.

About the City of Cambridge GIS Data

All of our data is in Massachusetts State Plane Coordinate System NAD 83, feet, except those files converted to GeoJSON which are in WGS84. We are constantly upgrading and improving our data layers and hopefully you will find this data useful and accurate. Please notify Cambridge GIS if you see any issues with the data by contacting us using this form.

City of Cambridge, MA, GIS Data Sets Available on GitHub

There are three ways to access the Cambridge GIS data on GitHub:

  1. You can access the cambridgegis_data repository directly to get the full data set.
  2. You can click on an individual category header below to get the repository for that category only.
  3. You can choose individual layers from the list below. Layers marked with an asterisk (*) are too large for GitHub's preview or individual download features, so they are available in the full or category repositories only. For layers marked with two asterisks (**), the GeoJSON version is too large but the TopoJSON is not, so the link will take you to the TopoJSON version.
